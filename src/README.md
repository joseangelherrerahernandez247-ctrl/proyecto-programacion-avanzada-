# Cerradura Digital 🔐

## Descripción
Este proyecto consiste en una cerradura digital básica que permite ingresar una clave para activar un LED o servo.

## Funcionamiento
1. El usuario ingresa una clave.
2. El sistema la compara.
3. Si es correcta, se activa el LED.
4. Si es incorrecta, no hace nada.

## Requisitos
- fpga 
- LED
- Resistencias
- Cables

## Ejecución
1. Cargar el código al dispositivo
2. Abrir el monitor serial
3. Ingresar la clave (ejemplo: 1234)
